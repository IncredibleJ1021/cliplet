# 变更日志

## 0.4.0

- 选择剪切板历史项后，默认会恢复原前台应用并自动粘贴，减少“先复制、退出面板、再手动粘贴”的步骤。
- 新增自动粘贴偏好设置、辅助功能权限状态和授权入口；无权限时会安全退回为只复制到剪切板。
- 优化历史面板视觉：使用系统弹出面板材质、更紧凑的列表行、圆角选中态和更清晰的文本/图片缩略图。
- 更新 README 中的 npm 配置示例和当前功能范围说明。

## 0.3.2

- 将应用图标替换为简约水墨风图标，并作为仓库内显式资源参与打包。

## 0.3.1

- 新增应用图标，避免 DMG 和 Finder 中显示通用占位图标。
- 优化 DMG 安装盘视觉：放大图标、固定窗口布局，并新增拖拽安装提示背景。
- 新增 npm 安装器配置，可通过 npm/npx 下载 GitHub Release 并安装到 Applications。

## 0.3.0

- 保留文本剪切板内容的首尾空白和换行，确保历史项可原样恢复。
- 将图片剪切板数据迁移到 Application Support 文件存储，历史记录仅保存元数据和文件引用。
- 收紧剪切板历史数据模型，减少文本/图片条目的非法状态。
- 修复从历史面板复制条目后被剪切板监控重复捕获的问题。
- 全局快捷键注册失败时不再静默保存无效设置，并在偏好窗口中提示用户。

## 0.2.0

- 新增图片剪切板历史。
- 历史面板新增图片缩略图和图片大小信息。
- 升级到文本/图片混合数据模型时，保留已有文本历史。

## 0.1.2

- 修复其他控件聚焦时无法录制全局快捷键的问题。
- 为应用窗口新增 `⌘W` 关闭窗口和 `⌘Q` 退出应用支持。

## 0.1.1

- 将应用和发布产物重命名为 `cliplet`。
- 将 `AGENTS.md` 改为中文。

## 0.1.0

- 初始菜单栏剪切板历史应用。
- 新增可配置的全局快捷键和历史保存数量。
- 新增本地打包脚本和 GitHub 发布流程。
